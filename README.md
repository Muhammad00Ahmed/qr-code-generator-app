# QR Code Generator App

QR code generator with customizable colors, sizes, and error correction levels.

## Features

- Generate QR codes from text/URLs
- Customize colors and sizes
- Error correction levels
- Download as PNG/SVG
- Batch generation
- Contact information encoding
- WiFi QR codes
- vCard support

## Tech Stack

- HTML5
- CSS3
- JavaScript
- QRCode.js library

## Usage

Open `index.html` in your browser and start generating QR codes!
